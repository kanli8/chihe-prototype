# AI 口腔门诊智能坐席 Demo

一个基于 AI 大模型的智能客服坐席系统演示页面。

## 功能特性

- 🤖 **AI 对话**：基于 OpenAI 的智能问答
- 📚 **知识库配置**：上传文档构建知识库
- ⚙️ **触发规则**：时间+事件组合触发自动消息
- 💬 **双视角展示**：坐席控制台 + 患者对话窗口
- 📊 **任务日志**：自动记录发送历史
- 👥 **患者管理**：切换不同患者查看对话
- 🌈 **多主题**：支持深色、浅色、科技蓝、清新绿主题

## 快速开始

1. 打开 `index.html`
2. 在底部输入框配置你的 OpenAI API Key
3. 开始使用

## 部署

支持部署到：
- Cloudflare Pages
- GitHub Pages
- 任意静态托管服务

## 技术栈

- HTML5 + CSS3 + JavaScript
- OpenAI API
- Cloudflare Pages (部署目标)
